# Projeto de Automação - Hub de Leitura (EBAC)

Este projeto contém a automação de testes de interface para a aplicação Hub de Leitura, utilizando Cypress e a biblioteca Faker para geração de massa de dados fictícios.

## 🚀 Tecnologias Utilizadas

- [Cypress](https://www.cypress.io/)
- [Node.js](https://nodejs.org/)
- [Faker.js](https://fakerjs.dev/)

## 🛠️ Como rodar o projeto

1. Clone o repositório:
   `git clone https://github.com/joaomarceloQA/teste-ebaq-ui.git`
2. Instale as dependências:
   `npm install`
3. Abra o Cypress:
   `npx cypress open` ou `npx cypress run` para rodar em modo headless.

## 🧪 Cenários Testados

- Cadastro de usuário com dados dinâmicos (Faker).
- Validação de estrutura da Home em ambiente local.
